# About this project

This project showcases the development of my personal website, 
which combines my portfolio and blog in a single cohesive platform.
The website was designed and implemented based on a 
comprehensive design system, ensuring consistency and cohesion throughout.

<SpacerS />

The goal was to create 
a digital presence that reflects my skills as both a **designer** and a **developer**, with 
a focus on clean aesthetics, intuitive navigation, and interactive elements.

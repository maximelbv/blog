## What are snippets ?

Code snippets are small blocks of reusable code that you can quickly insert into your programs.

In VS Code, snippets help developers avoid repetitive tasks by providing a quick way to generate
common code structures.

Snippets can be accessed through VS Code extensions, many popular programming languages have dedicated extensions that come with pre-defined snippets.

If you want a bundle of pre-defined snippets, search for 'snippets' followed by the language you go with, you'll find what you're looking for

or you can create your own custom snippets for personalized use.
